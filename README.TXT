TIMESTAMP MICROSERVICE

OVERVIEW

This is a simple microservice built with Node.js and Express that returns a timestamp in both Unix and UTC formats. 
If no date is provided, it returns the current time. If an invalid date is provided, it returns an error.

FEATURES
    •    Returns the current timestamp in Unix and UTC format

    •    Accepts date strings and Unix timestamps as input

    •    Handles invalid date inputs gracefully

PREREQUISITES
    •    Node.js installed

INSTALLATION
    •    Clone the repository

    •    Run npm install to install dependencies

    •    Start the server using npm start or node index.js

PROJECT STRUCTURE
    •    index.js – Main server file containing route logic

    •    package.json – Project metadata and dependencies
